# O básico sobre React

Este tutorial tem como objetivo apresentar os principais conceitos utilizados em uma aplicação React: JSX, components, props, state, lifecycle, hooks, etc.


## Sobre o React

React é uma biblioteca JavaScript utilizada para construir  interfaces de usuário.

React é uma das bibliotecas javascript mais populares, portanto tem uma comunidade gigante e muito material didático disponível.

React não é um framework (ao contrário do Angular, por exemplo, que é mais opnativo).

React é um projeto open source criado pelo Facebook.

Um dos aspectos mais importantes do React é o fato de que você pode criar componentes, que são como elementos HTML personalizados e reutilizáveis, para construir interfaces de usuário de maneira rápida e eficiente. O React também agiliza como os dados são armazenados e tratados, utilizando state, props, entre outras coisas.
